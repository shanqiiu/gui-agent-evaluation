# GUI Agent 执行轨迹自动判定

> 项目定位：构建针对 AI 智能体的自动化测试评估系统，实现对多模态 GUI Agent 任务执行合理性、结果正确性、执行效率的自动判定。

---

## 项目背景

随着多模态 GUI Agent（如手机操作助手）的快速发展，Agent 在实际执行任务时存在不确定性——可能产生频繁点击、重复操作、闪退、错误分支等异常行为。当前缺乏系统化的方法来捕获和评估这些执行故障点。

本项目旨在：
- **执行轨迹合理性判定**：基于操作前后截图，检测 Agent 操作是否出现异常
- **结果正确性判定**：不局限于"是否走到终点"，综合关键检查点达成率
- **执行效率判定**：通过语义相似性分析检测重复操作和无效操作
