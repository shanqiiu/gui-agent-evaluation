GUI Agent 测试判定——工作进展概要
技术基座
参考达尔文实验室的项目能力：基于 Qwen-VL-7B 微调，提供 GUI Agent 执行轨迹的功能完备性判定与功能可用性判定两项核心能力，作为本项目的技术起点。

调研进展
追踪 17 篇 2024-2026 顶会论文（ICML 2026、AAAI 2026、CVPR 2026），核心收获取下：

RoTS（ICML 2026 Spotlight）：11 类 Error Taxonomy，执行层 UI 误操作占比 42.9%，规划层恢复率 <20%
HalluClear：8 类 GUI 专属幻觉分类，证实感知幻觉是级联失败的根因
Completion-Correctness Gap：Agent 完成率 96% 但正确率远低于此，验证了项目"不依赖自报、客观判定检查点达成"的路线
项目提出的差分偏差三分类（无影响/补救性/级联偏差）已得到上述学术研究的验证。

技术规划
在达尔文实验室两项判定能力之上，扩展为五模块评测体系：

任务分解引擎（LLM + RAG 生成结构化检查点）
检查点验证器（VLM 截图语义对齐）
效率分析器（Embedding 检测重复/无效/循环操作）
轨迹差分判定器（三分类偏差判定）
综合评估器（加权聚合输出报告）
技术栈：Qwen3.5-VL-7B + Qwen3-8B/GPT-4o + BGE-M3 + ChromaDB/LanceDB。
